# 18 VICTOIRE

Numéro de fonction : 18

Nom de la fonction : 18. Victoire

**L'AGRESSEUR EST VAINCU.**

(Définition : victoire. Désignation : I)

1. L'agresseur est battu en combat ouvert (I¹).

2. Il est vaincu dans une compétition (I²).

3. Il perd aux cartes (I³).

4. Il perd à la pesée (I⁴).

5. Il est tué sans combat préliminaire (I⁵). Un dragon est tué pendant son sommeil (141). Zmialan se cache dans le creux d'un arbre ; il est tué (164).

6. Il est banni directement (I⁶). Une princesse, possédée par un diable, place une image sacrée autour de son cou : "Le pouvoir maléfique s'envola en un nuage de fumée" (115).

La victoire se rencontre aussi sous une forme négative. Si deux ou trois héros sont partis au combat, l'un d'eux (un général) se cache, tandis que l'autre est victorieux (désignation : *I¹).
