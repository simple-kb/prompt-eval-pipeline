# 20 RETOUR

Numéro de fonction : 20

Nom de la fonction : 20. Retour

**LE HÉROS RETOURNE.**

(Définition : retour. Désignation : ↓)

Un retour s'accomplit généralement par les mêmes formes qu'une arrivée. Cependant, il n'est pas nécessaire d'attacher une fonction spéciale à suivre un retour, puisque retourner implique déjà un franchissement de l'espace. Cela n'est pas toujours vrai dans le cas d'un départ. Après un départ, un agent est donné (un cheval, un aigle, etc.) et puis le vol ou d'autres formes de voyage se produisent, tandis qu'un retour a lieu immédiatement et, pour la plupart, sous les mêmes formes qu'une arrivée. Parfois le retour a la nature d'une fuite.
