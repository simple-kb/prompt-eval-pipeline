# 24 IMPOSTURE

Numéro de fonction : 24

Nom de la fonction : 24. Imposture

**UN FAUX HÉROS PRÉSENTE DES PRÉTENTIONS NON FONDÉES.**

(Définition : prétentions non fondées. Désignation : L)

Si le héros arrive chez lui, les fausses prétentions sont présentées par ses frères. S'il sert dans un autre royaume, un général, un porteur d'eau, ou d'autres les présentent. Les frères se font passer pour les ravisseurs du prix ; le général se fait passer pour le vainqueur d'un dragon. Ces deux formes peuvent être considérées comme des classes spéciales.
